# My Portfolio

This repository hosts a personal portfolio website built with **React** and **Vite**. It serves as a landing page to showcase projects, skills and contact information. The design follows a clean, modern layout provided `Landing Page.png` built using Figma.

## Features

- Responsive hero section with an introduction and call-to-action
- Feature list highlighting key technologies or services
- Testimonials / project showcase area
- Sticky header and footer with navigation links
- Modal support (via context) for popups or dialogs

## Structure

Components are organized under `src/components` with each component in its own folder alongside its CSS file. Custom hooks live in `src/hooks` and utilities/constants in `src/utils`. Global state is managed via contexts located in `src/context`.
